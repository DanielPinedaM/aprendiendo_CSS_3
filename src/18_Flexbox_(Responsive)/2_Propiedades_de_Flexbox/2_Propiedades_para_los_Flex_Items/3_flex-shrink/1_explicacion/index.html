flex-shrink:

Determina que tanto pueden encogerse (contraerse) los Flex Items.

Cuando el Flex Container NO tenga espacio sobrante (disponible), los Flex Items podrán disminuir su tamaño dependiendo del valor de flex-shrink:

Articulo - Ejemplo de flex-shrink: Articulo - ¿Cómo se Calcula flex-shrink: ?

flex-shrink: 1;

Los Flex Items SI pueden reducir (encoger) su tamaño cuando el tamaño del Flex Container disminuye (valor por defecto)

en este caso como ambos Flex Items tienen el mismo valor de flex-shrink: 1;

entonces se encogeran lo mismo

flex-shrink: 1;

flex-shrink: 1;

flex-shrink: 0;

Los Flex Items NO pueden reducir (encoger) su tamaño

Esto va a generar una barra de scroll horizontal en la pagina web

y PUEDE (no siempre, depende del min-width: del Flex Container) hacer q los Flex Items se desborden

flex-shrink: 0;

flex-shrink: 0;

flex-shrink: 2; y flex-shrink: 4;

Cuanto mas GRANDE sea el numero de flex-shrink: MAS se pueden encoger los Flex Items

y cuanto mas MENOR sea el numero de flex-shrink: MENOS se pueden encoger los Flex Items

En este caso flex-shrink: 4; tiene el valor mas GRANDE (es el Flex Item mas pequeño)

y flex-shrink: 2; tiene el valor mas PEQUEÑO (es el Flex Item mas grande)

flex-shrink: 2; NO es el doble de flex-shrink: 4;

flex-shrink: 2;

flex-shrink: 4;
