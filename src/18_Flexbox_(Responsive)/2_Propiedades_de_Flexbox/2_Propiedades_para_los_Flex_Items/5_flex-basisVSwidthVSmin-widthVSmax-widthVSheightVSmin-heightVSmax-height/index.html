¿Cuál es la Prioridad Entre las Propiedades flex-basis: VS width VS min-width VS max-width VS height VS min-height VS max-height ?

Samantha Ming - flex-basis: VS width: VS min-width: VS max-width: Mastery Games

flex-basis: auto; flex-direction: row; width: 250px;

Este es su valor por defecto. Funciona como un condicional, cuando SI se ha definido un ancho width: y/o alto height: entonces este será el tamaño (medida) que tomaran los Flex Items Y cuando NO se ha definido un ancho width: NI alto height: entonces los Flex Items ocuparan el espacio de su contenido (imágenes, video, texto, etc.), es decir, que tendrá el mismo comportamiento de flex-basis: content;
