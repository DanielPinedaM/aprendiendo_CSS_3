Dar una Distancia de Separación (Espacio) Entre los Flex Items y el Flex Container

Usar margen margin: para separar los Flex Items es mala practica, lo correcto (buena practica) es usar gap: row-gap column-gap;

Relleno padding: se usa para dar una distancia ALREDEDOR de los Flex Items (una distancia entre el Flex Container y los Flex Items)

Margen margin: y relleno padding: Responsive

Si el viewport se estira o encoge entonces la distancia por la q estan separados los div tambien lo hara, para ello uso margin: y padding: en porcentaje %

Para que la distancia de separación entre los Flex Items y el Flex Container sea Responsive usar el margen margin: y relleno padding: en porcentaje % para que el Layout se adapte al viewport

margin: y padding: VS gap:

gap: se aplica a TODOS los Flex Items en cambio margin: y padding: sirven para CUALQUIER elemento.

Con margin: y padding: puedo seleccionar elementos (etiquetas) HTML en ESPECIFICO, lo cual no puedo hacer con gap: entonces voy a darle un margen hacia la derecha margin-right: 4rem; al Flex Item 2 y al Flex Container un relleno hacia abajo padding-bottom: 2rem;

En el resultado te das cuenta de que margin: y padding: me permiten dar una distancia de separación a CUALQUIER elemento (etiqueta) HTML

Usando todas las propiedades juntas (mala practica)

  • margen margin:
  • relleno padding:
  • gap:

CSS-Tricks (Trucos CSS)

Usar margin: y gap: juntos para los Flex Items es mala practica

Lo correcto es usar una de las dos propiedades ( margin: ó gap: ) porque cuando uso margin: y gap: juntos para separar los Flex Items entonces los Flex Items se separan SUMANDO la distancia de margin: MAS la de gap: y hacer esto no tiene sentido (es ilógico).

Lo que pasa a continuacion es lo siguiente, mira los colores para que diferencies cada una de las propiedades:

  • gap: da una distancia de separación ENTRE los Flex Items.
  • Margen margin: separa más (las distancias se suman) los Flex Items, agrega espacio a AMBOS LADOS (izquierdo y derecho) de los Flex Items.
  • Relleno padding: da un espacio DENTRO de los Flex Items.
