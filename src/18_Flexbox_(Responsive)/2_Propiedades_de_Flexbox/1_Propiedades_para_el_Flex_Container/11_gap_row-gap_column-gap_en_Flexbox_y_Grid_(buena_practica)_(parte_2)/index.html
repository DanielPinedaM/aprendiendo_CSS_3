gap: row-gap column-gap;

gap: es una propiedad abreviada (shorthand) que sirve en Flexbox, Grid y CSS Multiple-column Layout para definir cuál es la distancia de separación (espacio) de los Flex Items entre sus filas ↔ row-gap: y columnas ↕ column-gap:

A esto también se le llama “gutters” (canalones)

Su sintaxis es gap: row-gap column-gap;

gap: puede tomar como valores:

  • Numero con unidades de medida de CSS (px, em, rem, %, vw, vh, vmin, vmax), este número no puede ser negativo (no puede ser menor que cero), tiene que ser mayor o igual que cero (si puede ser cero, un número mayor a cero o decimal):
  • La función gap: calc();
  • Puede tener un solo valor. Ejemplo: gap: 4rem; o dos valores Ejemplo: gap 2rem 4rem;

gap: normal; (valor por defecto)

Su valor por defecto es gap: normal; para Flexbox y Grid y gap: 1em; para CSS Multiple-column Layout

gap: normal; es lo mismo que gap: 0; en Flexbox y Grid ambos significan que NO hay distancia de separación entre los Flex Items, por lo cual los Flex Items están juntos, en cambio en CSS Multiple-column Layout su valor por defecto es gap: 1em;

Si defino gap: en porcentaje % entonces tiene un comportamiento diferente para Flexbox y Grid.

En Grid row-gap: en porcentaje ocasiona desbordamiento (overflow) de los elementos div hijos, por eso para Grid debo definir row-gap: en rem, em ó px y column-gap: en Grid si se puede usar con porcentaje:

  • colum-gap: separa los Flex Items VERTICALmente ↕
  • row-gap: separa los Flex Items HORIZONTALmente ↔

gap: NO SIEMPRE es la distancia de separación entre los Flex Items

CSS-Tricks (Trucos CSS)

Esto se debe a que también hay otras propiedades que afectan como se separan los Flex Items.

gap: SI funciona (siempre) con align-items: pero NO SIEMPRE va funcionar con justify-content: y align-content:

Los valores de las propiedades align-items: align-content: y justify-content: con los que SI funciona gap: son:

  • stretch;
  • flex-start;
  • flex-end;
  • center;
  • baseline;

justify-content: flex-start; gap: 30px;

Tengo un Flex Container al que le definí gap: 30px; y justify-content: flex-start;

Los Flex Items SI se están separando por el gap: 30px; porque lo que hace justify-content: flex-start; es juntar los Flex Items uno al lado del otro (no genera un espacio entre los Flex Items):
